## Abstract 
&emsp;The main goal of this project is to design a web application that accepts users’ resumes in either PDF or Word DOCX files, analyzes the resume, provides a scoring system and feedback to the user on their resume while also providing recommendations regarding the possible career paths that the users can consider based on their resume. <br>
&emsp;Many job seekers currently struggle to understand which career paths align best with their skills and experience. Our website would save users a lot of time trying to search for a career in the workforce, give those who are unsure about their career path a personal recommendation for their future, and allow them to improve their resume.
